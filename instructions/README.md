# AI Workspace Instructions

This directory contains guidance for AI agents and prompt usage in the OWASP SIST workspace.

- `.github/copilot-instructions.md` — primary AI system guidance for Copilot and AI assistants
- `prompts/` — reusable prompt templates for UI, layout, and animation generation
- `agents/` — specialized instruction profiles for design, security, and frontend architecture
- `.vscode/mcp.json` — local MCP server configuration for workspace context integration

Use this workspace scaffold to keep AI outputs consistent, secure, and production-quality.
